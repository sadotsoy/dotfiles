# Tricks

## Delete

- Delete without overwriting the register --> `"_d{motion}`.


## Visual

- Navigate to the begining or end of visual selection text --> ``` `< ``` or ``` `> ```


## Indent

- Indent current line and next 3 --> `4==`
- Indent around paragraph --> `=ap`
- Indent to the end of method --> `=%`
